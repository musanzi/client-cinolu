# Module Update Plan: Mentor Dashboard - Cinolu.org

## Context:
We are updating the "resources" module to be directly integrated into the details section of mentored programs. This eliminates the need for it to be a separate, standalone option, improving usability and coherence.

## Tasks:
1. **Backend Adjustments**:
  - Update the API to retrieve and serve resources associated directly with mentored program details.
2. **Frontend Changes**:
  - Modify the `ProgramDetails` component to include the `resources` module directly.
  - Refactor `resources` layout to function seamlessly within the `ProgramDetails` context.
3. **Remove Standalone Module**:
  - Deprecate the current isolated `resources` module on the dashboard.

## To-Do:
- Ensure all APIs serving resources respect the new hierarchical context under `ProgramDetails`.
- Test changes extensively to confirm seamless integration and no regression.
